# PlayerBatch Video Edit Guide

This is an editing-focused guide for a `PlayerBatch` video with Sam.

It is not a voiceover script.
It is a practical checklist of:

- what to show
- what order to show it in
- what commands are worth putting on screen
- where the hype moments should happen

## Core Video Idea

The viewer should understand this fast:

`PlayerBatch lets you spawn and control giant armies of fake players in Minecraft.`

The edit should feel like:

- instant scale
- funny chaos
- "wait that mod actually does a lot"
- then bigger and bigger reveals

## Best Overall Structure

1. Huge hook
2. Basic spawning
3. Formation showcase
4. Wand / area summoning
5. Selection and control
6. Combat and targeting
7. Testing / stress / chaos
8. Best cinematic ending shot

## 1. Intro Hook

### What to show

- giant dense bot army immediately
- you and Sam reacting
- one fast cut of bots rushing or filling the arena

### Best visuals

- `/pb spawn 100 dense`
- giant overhead shot
- quick zoom into the crowd
- if possible, one dramatic angle from behind you and Sam

### What text to show on screen

- `100 FAKE PLAYERS`
- `ONE COMMAND`
- `THIS MOD IS INSANE`

### Editing note

Do not start with menus or explanation.
Start with scale.

## 2. Show The Basic Feature First

### Goal

Make the viewer instantly understand the mod before showing advanced stuff.

### What to show

- simple bot spawn
- random names / skins
- how fast they appear

### Commands to show on screen

```mcfunction
/pb spawn 20
/pb spawn 50
```

### Good edit idea

- split-screen or jump cuts showing `20`, then `50`, then `100`

## 3. Formation Showcase

### Goal

Show that this is not just random spawning.

### What to show

- `circle`
- `filled_circle`
- `dense`
- maybe `square`

### Commands

```mcfunction
/pb spawn 25 circle
/pb spawn 25 filled_circle
/pb spawn 64 dense
/pb spawn 36 square
```

### Editing note

Use quick match cuts:

- same location
- different formation each cut
- command flashes on screen briefly

## 4. Wand / Area Summoning

### Goal

This is one of the most visual features.

### What to show

- get the wand
- set point A
- set point B
- suddenly fill a whole area with bots

### Commands

```mcfunction
/pb wand
```

### Editing note

This part should feel like:

- "ohhhh you can paint bots into the world"

Good visual:

- first-person block click
- immediate cut to giant finished crowd

## 5. Selection And Control

### Goal

Show that you do not have to control bots one by one.

### What to show

- selecting all
- selecting a smaller count
- clearing selection
- glowing selected bots

### Commands

```mcfunction
/pb selection all
/pb selection count 10
/pb selection clear
```

### Editing note

Show glow or visual selection clearly so viewers understand the "working set" idea.

## 6. Combat And Targeting

### Goal

This is where the mod becomes a video, not just a utility.

### What to show

- enable combat
- make bots target you
- make bots target you and Sam
- show explicit targeting

### Commands

```mcfunction
/pb combat on
/pb target kill @a
/pb target clear
```

### Good moments to capture

- bots rushing one player
- bots splitting pressure between two players
- one player trying to escape while the other gets swarmed

### Editing note

This is a perfect place for:

- intense music
- fast cuts
- zooms on panic moments

## 7. Big Challenge Segment

### Goal

Give the video a challenge or escalation.

### Ideas

- can you and Sam survive 50 bots?
- can you survive 100 dense bots?
- can you escape after turning combat on?
- how many bots before the world starts dying?

### Commands

```mcfunction
/pb spawn 50 dense
/pb combat on
/pb target kill @a
```

### Editing note

This should be the funniest or most chaotic section.

Good stuff to keep:

- screaming
- accidental deaths
- getting cornered
- lag jokes if they happen
- betrayal / panic moments with Sam

## 8. Testing / "The Mod Is Actually Deep" Section

### Goal

Show that the mod is more than a meme.

### What to show

- full test command
- AI/pathing test feel
- proof that the mod has actual systems behind it

### Commands

```mcfunction
/pb test all
```

### Editing note

Keep this section shorter than the chaos stuff.
It should communicate depth, not stall the pace.

## 9. Safety / Recovery / Utility

### Goal

Show one or two practical commands so the mod feels usable.

### What to show

- canceling a dumb huge summon
- maybe repairing tags if needed

### Commands

```mcfunction
/pb cancel
/pb repair tags
```

### Editing note

This works well as a quick joke:

- "and yes, there is a panic button"

## 10. Final Cinematic Ending

### Goal

End on the coolest-looking shot in the whole video.

### Best ending ideas

- you and Sam in front of a giant dense army
- sunset / arena angle
- bots packed across a giant platform
- one last insane overhead shot

### Best command setups for final visuals

```mcfunction
/pb spawn 100 dense
/pb spawn 200 dense
```

### Editing note

If the video is comedy-heavy, end on:

- a funny disaster

If the video is hype-heavy, end on:

- a big cinematic crowd shot

## Commands Worth Showing On Screen

These are the best commands to actually surface visually in the edit:

```mcfunction
/pb spawn 20
/pb spawn 50 circle
/pb spawn 100 dense
/pb wand
/pb selection all
/pb selection count 10
/pb combat on
/pb target kill @a
/pb target clear
/pb test all
/pb cancel
```

## Best Pacing Advice

- first 30 seconds should be scale + chaos
- explain with visuals, not long talking
- keep commands on screen briefly, not for too long
- use the wand and dense formation sections as visual payoff moments
- keep the "deep systems" section shorter than the "funny/insane" section

## Best Thumbnail Shots To Capture While Recording

- giant dense bot army from above
- you + Sam in foreground, bot army in background
- bots rushing both of you
- massive packed crowd filling a platform

## Strong Video Angle Options

Pick one main framing for the edit:

- `We Spawned an Army of Fake Players in Minecraft`
- `This Mod Lets You Control 100 Fake Players`
- `Can We Survive a Bot Army?`
- `Minecraft But We Command a Fake Player Army`

## Short Cutdown Version

If you want to make a short promo clip too, use this order:

1. giant dense army shot
2. command on screen: `/pb spawn 100 dense`
3. quick wand clip
4. bots rushing players
5. `/pb target kill @a`
6. panic / funny reaction
7. final cinematic crowd shot

## Final Reminder

The video should not feel like:

- a devlog
- a command tutorial
- a wiki page in video form

It should feel like:

- "we found a mod that lets us control an army"
- then you prove it with escalating chaos
